### "I clicked start and nothing happened."
- **Status Check:** Look at the top right `SYSTEM:` indicator. Does it change to `WRITING...` or `RENDERING...`?
- **Network Latency:** You are connecting to the Pollinations Cloud API. If you are on weak 4G/5G, the first image may take up to 20 seconds.
- **Audio Context:** Tap the screen again anywhere. Sometimes Android requires two taps to unlock the Web Audio API synthesizer.

### "There is no sound."
- **Check the Switch:** On iPhone, the physical "Silent" switch on the side MUST be off.
- **Check the Output:** Ensure "Media Volume" is up, not just "Ringer Volume."

### "The image is static."
- The Platinum Edition uses a **"Cinematic Pan"** (Ken Burns effect) instead of choppy AI video. This allows for 4K resolution. If it isn't moving, your browser's "Low Power Mode" might be disabling CSS animations. Turn off Battery Saver.
